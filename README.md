1. Program ini digunakan untuk mengecek apakah suatu bilangan termasuk bilangan prima atau bukan.
Pertama, program meminta input satu bilangan dari pengguna.
Jika bilangan kurang dari atau sama dengan 1, maka langsung dinyatakan bukan prima.
Jika lebih dari 1, program akan melakukan perulangan dari angka 2 sampai akar dari bilangan tersebut.
Jika ditemukan bilangan yang bisa membagi habis, maka bilangan tersebut bukan prima.
Jika tidak ditemukan, maka bilangan tersebut adalah bilangan prima.

<img width="956" height="450" alt="image" src="https://github.com/user-attachments/assets/36499774-0fae-4392-9063-6accc7c25a2a" />
2. Program ini bertujuan untuk menentukan jumlah pecahan uang dari nominal yang dimasukkan pengguna.
Pecahan uang disimpan dalam array, dimulai dari pecahan terbesar sampai terkecil.
Program membagi jumlah uang dengan setiap pecahan untuk mengetahui berapa lembar yang dibutuhkan.
Jika hasil pembagian lebih dari nol, maka jumlah lembar akan ditampilkan.
Sisa uang akan dihitung menggunakan operasi modulus, lalu dilanjutkan ke pecahan berikutnya.

<img width="899" height="856" alt="image" src="https://github.com/user-attachments/assets/8a70aded-d0cd-40ac-a297-348e54d485cd" />
3. Program ini adalah kuis matematika yang menampilkan soal perkalian, pembagian, atau modulus secara acak.
Program menghasilkan dua bilangan acak dan satu operator acak.
Pengguna diminta menjawab hasil operasi tersebut.
Setelah pengguna menjawab, program langsung menampilkan jawaban yang benar.
Program akan terus berjalan selama pengguna memilih untuk melanjutkan kuis.

<img width="987" height="571" alt="image" src="https://github.com/user-attachments/assets/be88c72f-edfe-4079-9a1a-717732600958" />
4. Program ini digunakan untuk mengecek apakah dalam sebuah array terdapat elemen yang sama atau duplikat.
Program menggunakan dua perulangan bersarang untuk membandingkan setiap elemen array.
Jika ditemukan dua elemen dengan nilai yang sama, maka program menampilkan elemen tersebut sebagai duplikat.
Jika tidak ditemukan, maka program menampilkan bahwa tidak ada elemen duplikat.

5. Program ini adalah permainan lempar dadu antara komputer dan pemain.
Setiap ronde, komputer dan pemain sama-sama melempar dadu dengan nilai acak 1 sampai 6.
Nilai lemparan disimpan ke dalam ArrayList untuk mencatat riwayat permainan.
Pemenang setiap ronde ditentukan dari nilai dadu yang lebih besar.
Permainan akan berhenti jika salah satu mencapai 5 kemenangan.
Setelah selesai, program menampilkan pemenang dan riwayat lemparan.
